# 🤖 AI Assistant Guide - COVERF.LB

## 🎯 ما هو AI Assistant؟

AI Assistant هو مساعد ذكي تم إضافته لمساعدتك في حل مشاكل رفع الصور على ImgBB تلقائياً.

## 🚀 كيفية الاستخدام

### 1️⃣ الوصول إلى AI Assistant
- **الموقع:** يظهر في الزاوية العلوية اليمنى من الصفحة
- **الشكل:** أيقونة روبوت مع اسم "AI Assistant"
- **الحالة:** يظهر "Ready to help" عندما يكون جاهزاً

### 2️⃣ الأزرار المتاحة

#### 🔍 **Diagnose System** - تشخيص النظام
- **الوظيفة:** يفحص النظام بالكامل
- **يختبر:** API Key، الشبكة، المتصفح، الصلاحيات
- **النتيجة:** تقرير مفصل عن المشاكل

#### 🔧 **Fix ImgBB** - إصلاح ImgBB
- **الوظيفة:** يحاول إصلاح مشاكل ImgBB تلقائياً
- **يشمل:** إعداد API Key، فحص الشبكة، إصلاح المتصفح
- **النتيجة:** إصلاح تلقائي للمشاكل

#### 🧪 **Test Upload** - اختبار الرفع
- **الوظيفة:** يختبر رفع صورة تجريبية
- **يشمل:** إنشاء صورة اختبار، رفعها، عرض النتيجة
- **النتيجة:** تأكيد أن النظام يعمل

## 📋 خطوات حل المشاكل

### الخطوة 1: تشخيص النظام
1. **انقر على "Diagnose System"**
2. **راقب النتائج في السجل:**
   - ✅ **OK** = كل شيء يعمل
   - ❌ **Issue** = هناك مشكلة

### الخطوة 2: إصلاح المشاكل
1. **انقر على "Fix ImgBB"**
2. **إذا طلب API Key:**
   - اذهب إلى https://api.imgbb.com/
   - سجل حساب جديد
   - احصل على API Key
   - أدخله عندما يطلب منك

### الخطوة 3: اختبار النظام
1. **انقر على "Test Upload"**
2. **راقب النتيجة:**
   - ✅ **Success** = النظام يعمل
   - ❌ **Failed** = هناك مشكلة

## 🔧 إعداد API Key

### الحصول على API Key:
1. **اذهب إلى:** https://api.imgbb.com/
2. **انقر على "Sign up"**
3. **املأ البيانات المطلوبة**
4. **سجل الدخول**
5. **اذهب إلى "API" في القائمة**
6. **انسخ API Key**

### إدخال API Key:
1. **انقر على "Fix ImgBB"**
2. **أدخل API Key عندما يطلب منك**
3. **سيتم حفظه تلقائياً**

## 📊 قراءة السجل (Log)

### ألوان السجل:
- 🔵 **أزرق:** معلومات عامة
- 🟢 **أخضر:** نجح العملية
- 🟡 **أصفر:** تحذير
- 🔴 **أحمر:** خطأ

### أمثلة على الرسائل:
```
[10:30:15] AI Assistant started monitoring system... (أزرق)
[10:30:16] API key not configured properly (أحمر)
[10:30:17] API key saved successfully (أخضر)
[10:30:18] Test upload successful: https://i.ibb.co/xxx.jpg (أخضر)
```

## 🛠️ حل المشاكل الشائعة

### المشكلة: "API key not configured"
**الحل:**
1. انقر على "Fix ImgBB"
2. أدخل API Key الصحيح
3. اختبر النظام

### المشكلة: "Network is offline"
**الحل:**
1. تحقق من اتصال الإنترنت
2. أعد تشغيل الراوتر
3. جرب شبكة أخرى

### المشكلة: "Browser compatibility issues"
**الحل:**
1. استخدم متصفح حديث (Chrome, Firefox, Safari)
2. حدث المتصفح
3. امسح الكاش

### المشكلة: "All upload methods failed"
**الحل:**
1. تحقق من API Key
2. تحقق من اتصال الإنترنت
3. جرب في متصفح آخر
